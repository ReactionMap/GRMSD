フォルダ構成

algo   : RMSDを計算するアルゴリズム実装を格納したフォルダ
query  : 検索したい構造のcsvファイルたちを格納したフォルダ(同一ファイル内の分子は同一原子数)
target : 検索対象のcsvファイルたちを格納したフォルダ(同一ファイル内の分子は同一原子数)
sdf2csv   : sdfファイルをcsvファイルに変換するプログラムを格納したフォルダ
---------------------------------------------------------------------------------------------
calculate_RMSDs.mの使用方法

入力:
 options (省略可)

出力
 q_t_result.csvというファイルが生成されます.
 (q,tはquery,targetのファイル名が入ります)

変数の説明--------------------------------------------------
options : use_label,permit_mirror,ignore_atomの属性を持つ構造体.
各属性は指定しなければデフォルトの値が渡される

use_label     :RMSDの計算において原子種の情報を使うか否かの論理値.使う場合はtrue.
               (デフォルトはtrue)
permit_mirror :RMSDの計算において鏡映を許すか否かの論理値.許す場合はture
               (デフォルトはfalse)
ignore_atom   :RMSDの計算において考慮したくない原子の原子番号配列.
               (水素を考慮せずCO骨格のみでRMSDを計算させたい場合は[1]をわたすことになる)
               (デフォルトは[1],水素を無視する)
clus_mode     :trueならばqueryとtargetのファイル名が等しいときのみRMSDを計算します.
               (デフォルトはfalse)
reduction     :RMSDを計算するアルゴリズムにおいて計算量を減らすかどうかの論理値.減らす場合true.
               (デフォルトはtrue)
iter_num      :reductionがtrueのとき, 何回初期値を取り直すか.
               (デフォルトは4)
------------------------------------------------------------
結果のcsvファイルはi行j列目の値が
targetのi番目の分子と,queryのj番目の分子のRMSD
を表します.(計算中にエラーが出た場合はInfが割り当てられます)

----------------------------------------------------------------------------------------------
quey , targetに用いるcsvファイルの形式
nを１分子中の原子数として
先頭行     :  n,n,n,n
二行目以降 :  x座標,y座標,z座標,原子番号

(m種類の構造があるとき, 行数1+nm ,列数4のcsvファイルになります)
